Quick Start

Get up and running in minutes

1

Installation

git clone https://github.com/bezpieczniuczniowie-star/cactus.git
cd cactus
docker-compose up -d
2

Create API Key

curl -X POST http://localhost:3000/api/v1/keys \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"name": "My Key"}'
3

Send Email

curl -X POST http://localhost:3000/api/v1/messages/send \
  -H "Authorization: CTS_987654321098_ABCDEF_null" \
  -H "Content-Type: application/json" \
  -d '{
    "to": "user@example.com",
    "from": "noreply@yourdomain.com",
    "subject": "Hello!",
    "html": "
Welcome!
"
  }'
4

Track Delivery

Access your dashboard at http://localhost:3000 to monitor real-time delivery status, opens, clicks, and analytics.

API Examples

Simple REST API for all your email needs

Send Template Email

POST /api/v1/messages/send-template

{
  "to": "user@example.com",
  "template_id": "tpl_welcome",
  "variables": {

  }
}

Get Analytics
