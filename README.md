# CACTUS Mailer Website

Official website for CACTUS Mailer - Enterprise Email Service Platform

## Features

- 🎨 Modern, responsive design
- 📱 Mobile-friendly
- ⚡ Fast performance
- 🔒 Secure
- 📊 Feature showcase
- 💰 Pricing plans
- 📚 Documentation links
- 🎯 Call-to-action sections

## Structure

- `index.html` - Homepage
- `privacy.html` - Privacy Policy
- `terms.html` - Terms of Service
- `styles/main.css` - Main stylesheet
- `js/main.js` - JavaScript functionality

## Development

To develop locally:

```bash
# Run a local server
python -m http.server 8000

# Or using Node.js
npx http-server
```

Then visit `http://localhost:8000`

## Deployment

This website is deployed to GitHub Pages at: https://bezpieczniuczniowie-star.github.io/cactus

The `gh-pages` branch is automatically published.
