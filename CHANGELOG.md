# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-15

### Added
- Initial release of CACTUS Mailer
- Core email sending API with SMTP, SendGrid, and AWS SES support
- Email tracking (opens, clicks, bounces)
- API key management system with CTS_* format
- Real-time analytics and reporting dashboard
- Webhook event system with retry logic
- Email template engine with Handlebars support
- Multi-provider email delivery support
- Database migrations and seeding
- Rate limiting and security features
- Docker Compose deployment configuration
- React web dashboard
- Comprehensive API documentation
- Contributing guidelines
- MIT License

### Planned Features (Upcoming Releases)
- Desktop applications (Windows .exe, macOS .dmg)
- Mobile application (Android .apk)
- Advanced A/B testing
- SMS sending module
- Push notifications
- Advanced email segmentation
- AI-powered subject line optimization
- Multi-language support
- Blockchain email verification
- Enhanced GDPR compliance tools
